Comando para o cmd :
cd = trocar de diretorio 
dir = mostrar tds os diretorios
mkdir = criar um diretorio 
rmdir = remover um diretorio
cls = limpar prompt
tree = mostra a arvore de diretorios de uma forma grafica
echo = imprime no prompt
> = pega a saida de um comando e grava no arq indicado
type = ver oq tem dentro do arquivo
del = apagar arq
copy = cria um cópia do arq
move = move arquivo
rename = renomeia arq
code -n filename- Abre o arquivo em uma nova janela. 
code -r filename- Abre o arquivo na janela já aberta
